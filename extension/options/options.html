Notion databases

Database schema

The extension matches Notion properties by name. Properties not in your DB are silently skipped, so create only the ones you want. The title property in each DB is auto-detected — it doesn't need to be named "Name".

Recommended properties

Companies

  • Name — Title
  • Website — URL
  • Industry — Select or Text
  • Description — Text
  • Location — Text
  • Phone — Phone
  • Source URL — URL

Contacts

  • Name — Title
  • Title — Text
  • Email — Email
  • Phone — Phone
  • LinkedIn — URL
  • Company — Text
  • Source URL — URL
